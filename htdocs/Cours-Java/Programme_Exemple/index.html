Bienvenue dans l'exemple de programme en Java: on va construire un programme simple en Java en utilisant les concepts de base.

Création d'une classe principale

//Fichier: Principale.java
public class Principale {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Ce code affiche "Hello, World!" dans la console. C'est un exemple classique pour débuter en Java.

Ajout d'une classe supplémentaire

//Fichier: Personne.java
public class Personne {
    private String nom;
    private int age;

    public Personne(String nom, int age) {
        this.nom = nom;
        this.age = age;
    }

    public String getNom() {
        return nom;
    }
    public int getAge() {
        return age;
    }

    public void sePresenter() {
        System.out.println("Bonjour, je m'appelle " + nom + " et j'ai " + age + " ans.");
    }
}

Cette classe représente une personne avec un nom et un âge, et une méthode pour se présenter.

Utilisation de la classe Personne dans la classe Principale

//Fichier: Principale.java
public class Principale {
    public static void main(String[] args) {
        Personne personne = new Personne("Alice", 30);
        personne.sePresenter();
    }
}

Ce code crée une instance de la classe Personne et appelle sa méthode sePresenter().

Compilation et exécution

Pour compiler et exécuter ce projet Java, vous pouvez utiliser les commandes suivantes dans votre terminal :

javac Personne.java
java Principale

Assurez-vous que les fichiers Personne.java et Principale.java sont dans le même répertoire pour que la compilation fonctionne correctement.

Création d'une interface

//Fichier: Salutation.java
public interface Salutation {
    void saluer();
}

Cette interface définit une méthode saluer() que les classes peuvent implémenter pour fournir une salutation personnalisée.

Vous pouvez ensuite implémenter cette interface dans la classe Personne pour fournir une salutation spécifique :

//Fichier: Personne.java
public class Personne implements Salutation {
    private String nom;
    private int age;

    public Personne(String nom, int age) {
        this.nom = nom;
        this.age = age;
    }
    public String getNom() {
        return nom;
    }
    public int getAge() {
        return age;
    }
    public void sePresenter() {
        System.out.println("Bonjour, je m'appelle " + nom + " et j'ai " + age + " ans.");
    }
    @Override
    public void saluer() {
        System.out.println("Salut, je suis " + nom + " !");
    }
}

Maintenant, la classe Personne implémente l'interface Salutation et fournit une implémentation de la méthode saluer().

Utilisation des boucles

// Exemple d'utilisation d'une boucle for pour caluler la table de multiplication de 10
public class Table10 {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            System.out.println("10 x " + i + " = " + (10 * i));
        }
    }
}
