Structure Du Programme

  • Accueil

La structure habituelle d'un programme Java comprend plusieurs éléments clés, notamment les déclarations de classes, les méthodes, et les blocs de code. Un programme Java commence généralement par la déclaration d'une classe, qui est le point d'entrée du programme. À l'intérieur de cette classe, vous trouverez la méthode principale (main), qui est exécutée lorsque le programme est lancé. En plus de la méthode principale, une classe peut contenir d'autres méthodes pour organiser le code et faciliter la réutilisation.

Un projet est composé de plusieurs classes, chacune ayant une responsabilité spécifique. Habituellement, on crée une classe par fichier. Le projet doit être composé d'au moins une classe contenant la méthode main, qui est le point d'entrée du programme. Les autres classes peuvent être utilisées pour définir des objets, des fonctions, ou des structures de données. La classe contenant la méthode main est souvent appelée "classe principale" ou "classe d'exécution". C'est à partir de cette classe que le programme commence à s'exécuter, et elle peut appeler d'autres classes et méthodes pour accomplir des tâches spécifiques. Elle se rédige généralement de la manière suivante : 

//Main file
public class Main {
    public static void main(String[] args) {
        private Maclasse personne1 = new MaClasse("Jean", 25);

        personne1.afficherInformations();
    }
}

En dehors de la classe principale, pour créer une classe on suit souvent la structure suivante :

  • Déclaration de la classe : On commence par déclarer la classe en utilisant le mot-clé "class" suivi du nom de la classe.
  • Attributs : Ensuite, on peut définir les attributs de la classe, qui sont des variables qui stockent les données associées à chaque instance de la classe.
  • Constructeurs : Les constructeurs sont des méthodes spéciales utilisées pour initialiser les objets de la classe. Ils ont le même nom que la classe et n'ont pas de type de retour.
  • Getters et Setters : Ces méthodes permettent de lire et de modifier les attributs d'une instance de la classe de manière contrôlée.
  • Méthodes : Enfin, on peut définir des méthodes pour la classe, qui sont des fonctions associées à la classe et qui peuvent être appelées pour effectuer des actions spécifiques.

//Example of a class
public class MaClasse {
    // Attributs
    private String nom;
    private int age;

    // Constructeur
    public MaClasse(String nom, int age) {
        this.nom = nom;
        this.age = age;
    }

    // Getters et Setters
    public String getNom() {
        return nom;
    }

    public void setNom(String nom) {
        this.nom = nom;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    // Méthodes
    public void afficherInformations() {
        System.out.println("Nom : " + nom);
        System.out.println("Âge : " + age);
    }
}

Il existe trois types d'état pour un attribut : public, private, et protected. Un attribut public est accessible depuis n'importe quelle classe, tandis qu'un attribut private n'est accessible que depuis la classe dans laquelle il est déclaré. Un attribut protected est accessible depuis la classe dans laquelle il est déclaré et depuis les classes qui en héritent. Pour modifier la valeur d'un attribut on doit rajouter le terme this devant le nom de l'attribut, ce qui permet de différencier l'attribut de la classe de la variable locale utilisée dans le constructeur ou les méthodes.

Il existe des méthodes prédefinies en Java, comme toString() ou equals(), qui peuvent être utilisées pour effectuer des opérations spécifiques sur les objets. Voici une liste de ces méthodes les plus courantes :

Méthodes prédéfinies en Java pour les Strings
Méthode Description
var.toString() Retourne une représentation sous forme de chaîne de caractères de la variable var.
var.length() Retourne la longueur de la chaîne de caractères de la variable var.
var.charAt(int index) Retourne le caractère à l'indice spécifié dans la chaîne de caractères de la variable var.
var.substring(int beginIndex, int endIndex) Retourne une sous-chaîne de la chaîne de caractères de la variable var, depuis l'indice beginIndex jusqu'à l'indice endIndex.
var.equals(String chaine) Compare la variable var actuelle avec une autre variable pour vérifier s'ils sont égaux.
var.equalsIgnoreCase(String chaine) Compare la variable var actuelle avec une autre variable pour vérifier s'ils sont égaux, en ignorant les différences de casse.
var.contains(String chaine) Vérifie si la variable var contient une certaine chaîne de caractères.
var.toLowerCase() Retourne une nouvelle chaîne de caractères avec tous les caractères de la variable var convertis en minuscules.
var.toUpperCase() Retourne une nouvelle chaîne de caractères avec tous les caractères de la variable var convertis en majuscules.
Méthodes prédéfinies en Java pour les Integers et Doubles
Méthode Description
var.toString() Retourne une représentation sous forme de chaîne de caractères de la variable var.
var.intValue() Retourne la valeur entière de la variable var.
var.doubleValue() Retourne la valeur double de la variable var.
var.parseInt(String s) Retourne la valeur entière de la chaîne de caractères s.
var.parseDouble(String s) Retourne la valeur double de la chaîne de caractères s.
var.compareTo(int autre) Compare la variable var actuelle avec une autre variable pour vérifier si elle est inférieure, égale ou supérieure à l'autre variable.
Retourne un entier négatif, zéro ou un entier positif selon le résultat de la comparaison.
max(int a, int b) Retourne la valeur maximale entre les deux entiers a et b.
min(int a, int b) Retourne la valeur minimale entre les deux entiers a et b.
Méthodes prédéfinies en Java pour les Booleans
Méthode Description
var.toString() Retourne une représentation sous forme de chaîne de caractères de la variable var.
var.booleanValue() Retourne la valeur booléenne de la variable var.
var.parseBoolean(String s) Retourne la valeur booléenne de la chaîne de caractères s.
Méthodes prédéfinies en Java pour les Arrays
Méthode Description
var.length Retourne la longueur de l'array var.
Arrays.toString(var) Retourne une représentation sous forme de chaîne de caractères de l'array var.
Arrays.sort(var) Trie l'array var dans l'ordre croissant.
Méthodes prédéfinies en Java pour les List et ArrayList
Méthode Description
var.size() Retourne la taille de la collection var.
var.isEmpty() Retourne true si la collection var est vide, sinon retourne false.
var.contains(Object o) Retourne true si la collection var contient l'objet o, sinon retourne false.
var.add(Object o) Ajoute l'objet o à la collection var.
var.remove(Object o) Supprime l'objet o de la collection var s'il est présent.
var.remove(int index) Supprime l'élément à l'index spécifié de la collection var.
var.get(int index) Retourne l'élément à l'index spécifié de la collection var.
var.set(int index, Object o) Remplace l'élément à l'index spécifié de la collection var par l'objet o.
var.clear() Supprime tous les éléments de la collection var.
