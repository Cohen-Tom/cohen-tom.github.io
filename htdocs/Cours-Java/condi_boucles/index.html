Conditions

En Java, les conditions permettent de contrôler le flux d'exécution du programme en fonction de certaines conditions. Les structures de contrôle les plus courantes sont les instructions if, else if, et else. 

if (condition) {
    // Code à exécuter si la condition est vraie
} else if (autreCondition) {
    // Code à exécuter si l'autre condition est vraie
} else {
    // Code à exécuter si aucune des conditions précédentes n'est vraie
}

Pour mettre des opérateurs logiques dans les conditions, on peut utiliser les opérateurs suivants : && (ET logique), || (OU logique), et ! (inverseur). Par exemple : 

if (condition1 && condition2) {
    // Code à exécuter si les deux conditions sont vraies
}
if (condition1 || condition2) {
    // Code à exécuter si au moins une des conditions est vraie
}
if (!condition) {
    // Code à exécuter si la condition est fausse
}

Il est aussi possible d'utiliser des structures de contrôle plus complexes, comme les instructions switch, qui permettent de gérer plusieurs cas en fonction de la valeur d'une variable. 

switch (variable) {
    case valeur1:
        // Code à exécuter si la variable est égale à valeur1
        break;
    case valeur2:
        // Code à exécuter si la variable est égale à valeur2
        break;
    default:
        // Code à exécuter si la variable ne correspond à aucune des valeurs précédentes
}

Boucles

Les boucles permettent de répéter un bloc de code tant qu'une condition est vraie. Les types de boucles les plus courants en Java sont la boucle for, la boucle while, et la boucle do-while. 

for (initialisation; condition; incrémentation) {
    // Code à exécuter à chaque itération
}

while (condition) {
    // Code à exécuter tant que la condition est vraie
}

do {
    // Code à exécuter au moins une fois, puis répété tant que la condition est vraie
} while (condition);

La partie initialisation de la boucle for s'écrit de cette façon : int var = entier.
Les boucles peuvent également être imbriquées, ce qui permet de créer des structures de contrôle plus complexes. Par exemple, une boucle for à l'intérieur d'une boucle while.
