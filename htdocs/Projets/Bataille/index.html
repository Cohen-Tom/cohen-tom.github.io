Introduction

Le projet Bataille est une application Python qui reproduit le jeu de cartes classique de la bataille. Il oppose un joueur humain à un adversaire automatisé via une interface graphique complète.

Objectifs du projet

  • Développer un jeu de cartes interactif
  • Utiliser Tkinter pour l’interface graphique
  • Mettre en place un bot automatisé
  • Ajouter un système de paris et de gestion d’argent

Architecture générale

Le projet est structuré selon une architecture orientée objet. Le fichier Bataille.py centralise la logique du jeu, tandis que robot.py gère le comportement de l’adversaire.

Fonctionnement du jeu

Les cartes sont mélangées et distribuées équitablement. À chaque tour, le joueur et le bot jouent une carte. La carte de valeur la plus élevée remporte le pli.

Le jeu propose un mode manuel et un mode automatique, permettant d’enchaîner plusieurs coups successifs.

Gestion du bot

Le bot est implémenté dans le fichier robot.py. Il joue automatiquement la carte située au sommet de son paquet et applique strictement les règles du jeu de bataille.

Technologies utilisées

Technologie Rôle

Conclusion

Le projet Bataille met en œuvre une application complète mêlant logique de jeu, interface graphique et automatisation. Il constitue une base solide pouvant être enrichie par de nouvelles fonctionnalités.
