Commencer le code :                                          <!Doctype HTML>

----------Fonction-------------------------------------------------------Emplacement_minimum----------------------------------------------------------------------------------------------------------------
-> Creer une page web :                               <html> ... </html>
-> Ajouter l'option des accents :                 <meta charset='utf-8'> | (se met dans le head)
-> Mettre une place pour un nom a l'onglet :          <head> ... </head> | (se met dans le html)
-> Mettre un nom dans l'onglet :                  <title> texte </title> | (se met dans le head)
-> Rajouter du texte dans la page :                 <body> texte </body> | (se met dans le html, apres le head)
-> Creer un paragraphe :                                  <p> texte </p> | (se met dans le body)
-> Creer une mise en page, un ordre :                   <li> texte </li> | (se met dans le body)
-> Faire un alinea :                                    <o1> texte </o1> | (se met dans le body)
-> Creer un titre ou un sous-titre (n est un numero) :  <hn> texte </hn> | (se met dans le body)
n peut aller de 1 a 6
-> Creer une redirection :  <a href="lien_URL_ou_Lien_Fichier">texte</a> | (se met dans le body)
-> Retour a la ligne :                                              <br> | (se met dans le body)

Eviter la mise en page sur HTML :
-> mettre du texte en gras :                           <b>texte gras</b> | (se met dans le body)
-> mettre du texte en italique :                   <i>texte italique</i> | (se met dans le body)
-> mettre du texte en gros :                       <big>texte gros</big> | (se met dans le body)
-> mettre du texte en petit :                 <small>texte petit</small> | (se met dans le body)

Faire des liens avec d'autres types de fichier :
-> Faire un lien vers un fichier.css :      <link rel='stylesheet' href = 'fichier.css' | (se met dans le head)
-> Inserer une image :               <img src="URL/image.png" alt="Ceci est une image"> | (se met dans le body)

Faire des Tableaux:
-> Creer un tableau :                                                   <table></table> | (se met dans le body)
-> Rajouter une ligne :                                                       <tr></tr> | (se met dans table)
-> Rajouter une case :                                                   <td> Nom </td> | (se met dans tr)

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Structure minimum d'un fichier html:

<!Doctype HTML>

<html>
	<head>
    	<meta charset='utf-8'>
        <title>Nom de l'onglet</title>
    </head>
	
    <body>
    	...
        Code
        ...
    </body>
</html>
